## How to Run

```bash
# 1. Edit the configuration file
edit config.txt

# 2. Navigate to the project directory
cd directory

# OPTIONAL
mingw32-make clean

# 3. Build the project using MinGW Make
mingw32-make
# or
make

# 4. Run the executable
./csopesy.exe
```
